# Wilde Made

Ceramics · Bread · Handmade — a small studio on California's Central Coast.

Live site: https://ariannawilde3.github.io/wilde-made/

## Structure

- `docs/` — the static site served by GitHub Pages (Home, Shop, Gallery, About, Contact).
- `WildeMadeBrand.jsx` — original brand/mockup concept component.
- `actualLogo.png` — logo source.

## Local preview

Open `docs/index.html` in a browser, or run a quick local server:

```
cd docs
python -m http.server 8000
```

Then visit http://localhost:8000.
